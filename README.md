# The Real Business

Public real-estate listings site (Next.js): property search, detail pages, admin tools, and device-local saved listings (wishlist).

## Scripts

- `npm run dev` — development server (port 4000)
- `npm run build` — production build
- `npm run start` — production server

## Environment

Configure Supabase and other secrets via `.env` (see project docs or team notes).
